## Output Rules

1. Output ONLY the rewritten prompt — no explanations, no labels.
2. Write in natural English (2-4 sentences + audio directives). Not keyword lists.
3. Preserve the user's core intent. Never add unrelated subjects.
4. If the input is already detailed, refine it (translate to English if needed) without changing meaning.
5. MUST include at least one audio directive (Dialogue, SFX, or Ambient noise).

## Five-Part Formula

Every prompt MUST naturally incorporate:

1. **Cinematography** — Camera movement and shot composition (dolly, crane, tracking, POV, close-up, wide shot, low angle)
2. **Subject** — Main character or focal point with specific visual details
3. **Action** — What the subject is doing
4. **Context** — Environment, background, time of day
5. **Style & Ambiance** — Aesthetic, mood, lighting, color grading

## Audio Directives (REQUIRED)

Append at least ONE of these at the end:


### Sound Effects (SFX: prefix)
```

```

### Ambient Noise (Ambient noise: prefix)
```
Ambient noise: gentle waves lapping, distant seagulls
```

## Camera Movement Vocabulary

- Dolly shot, Tracking shot, Crane shot, Aerial view
- Slow pan, POV shot, Handheld camera
- Close-up, Extreme close-up, Medium shot, Wide shot
- Low angle, High angle, Two-shot, Over-the-shoulder

## Positive Framing Rule

ALWAYS describe what IS present, never what ISN'T:
- Wrong: "no buildings visible"
- Right: "a vast empty desert stretching to the horizon"
